## Experiment 8: Temperature Converter
**Path:** `exp 8/`

### Description
A simple Android application that converts temperature between Celsius and Fahrenheit.

### Features
- **Celsius to Fahrenheit:** Converts the input value from Celsius to Fahrenheit using the formula `(C * 9/5) + 32`.
- **Fahrenheit to Celsius:** Converts the input value from Fahrenheit to Celsius using the formula `(F - 32) * 5/9`.
- **Clear Functionality:** A button to reset the input and result fields.

## Experiment 9: Gender Selection (Radio Buttons)
**Path:** `exp9/`

### Description
An application demonstrating the use of `RadioGroup` and `RadioButton` widgets for single-choice selection.

### Features
- **Options:** Includes Male, Female, and Other options.
- **Selection Handling:** Uses `RadioGroup` to ensure only one option is selected at a time.
- **Display Result:** Shows the selected gender in a `TextView`.

## Experiment 10: WebView Implementation
**Path:** `exp10/`

### Description
An application that demonstrates how to embed a web browser within an Android app using `WebView`.

### Features
- **URL Loading:** Loads a specified website (e.g., YouTube) directly within the app.
- **Back Navigation:** Overrides `onBackPressed` to navigate back through history.

## How to Run
