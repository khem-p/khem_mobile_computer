# HelloWorld (App2)

This is a basic Android application that displays "Hello World!" on the screen. It's a standard starter project used to understand the structure of an Android app.

## Features
- Displays a `TextView` with the text "Hello World!".
- Uses **Edge-to-Edge** layout for a modern UI experience.
- Uses **Kotlin** as the primary programming language.

## Project Structure
- `MainActivity.kt`: The main entry point of the app.
- `activity_main.xml`: The layout file defining the UI components.

## How to Run
1. Open the project in Android Studio.
2. Build and run the app.
3. Observe the "Hello World!" message centered on the screen.
