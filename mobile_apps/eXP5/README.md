# eXP5

This Android application demonstrates the use of a `ToggleButton` and how to handle its state changes using a listener.

## Features
- **Toggle Button**: A simple interactive UI element.
- **Toast Notifications**: Displays a temporary message when the toggle state changes.
- **Event Handling**: Implements `OnCheckedChangeListener` to detect the "ON" or "OFF" state.

## Project Structure
- `MainActivity.java`: Contains the logic for the toggle button and its associated `Toast` messages.
- `activity_main.xml`: Contains the `ToggleButton` used for interaction.

## How to Run
1. Open the project in Android Studio.
2. Build and run the app.
3. Toggle the button to see the "ON" or "OFF" message on the screen.
