# SumCalculator (App4)

This Android application provides a simple calculator interface for basic arithmetic operations.

## Features
- **Arithmetic Operations**: Addition, Subtraction, Multiplication, and Division.
- **Input Validation**: Checks for empty inputs and valid numerical values.
- **Reset Button**: Clears all input fields and resets the result to its default state.
- **User Feedback**: Displays error messages if the input is invalid or if there is a division by zero.

## Project Structure
- `MainActivity.java`: Contains the main logic for all arithmetic operations.
- `activity_main.xml`: Defines the layout, including two `EditText` for inputs and various `Button` objects for calculations.

## How to Run
1. Open the project in Android Studio.
2. Build and run the app.
3. Enter two numbers and click on any operation button to see the result.
