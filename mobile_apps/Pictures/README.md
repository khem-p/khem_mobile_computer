# Pictures (Prac3)

This Android application features an automated image slideshow that cycles through a set of pre-defined images.

## Features
- **Automated Slideshow**: Automatically switches images every 3 seconds.
- **Image List**: Includes multiple images (`image1`, `image2`, `image3`, `image4`).
- **Resource Management**: Uses a `Handler` and `Runnable` to manage the slideshow cycle.

## Project Structure
- `MainActivity.java`: Contains the logic for the slideshow, handling image transitions, and lifecycle management.
- `activity_main.xml`: Contains the `ImageView` that displays the current slide.

## How to Run
1. Open the project in Android Studio.
2. Build and run the app.
3. Observe as the images cycle automatically on the screen.
